Keywords AI MCP Server

Model Context Protocol server for accessing Keywords AI data from your AI assistant

Features

  • Logs - Query and filter LLM request logs with powerful filtering
  • Traces - View complete execution traces with span trees
  • Customers - Access customer data and budget information
  • Prompts - Manage prompt templates and versions

Quick Start

Mode 1: Local Stdio (Personal Use)

Run locally for development - simplest setup:

git clone https://github.com/Keywords-AI/keywordsai-mcp.git
cd keywordsai-mcp
npm install && npm run build

Configure your MCP client (~/.cursor/mcp.json):

{
  "mcpServers": {
    "keywords-ai": {
      "command": "node",
      "args": ["/path/to/keywordsai-mcp/dist/lib/index.js"],
      "env": {
        "KEYWORDS_API_KEY": "your_api_key"
      }
    }
  }
}

Mode 2: HTTP Streaming - Private Deployment

Deploy to Vercel with API key stored as environment variable. No client-side key needed:

  1. Deploy: vercel deploy --prod
  2. Set KEYWORDS_API_KEY in Vercel Dashboard â†’ Environment Variables

Configure your MCP client:

{
  "mcpServers": {
    "keywords-ai": {
      "url": "https://your-project.vercel.app/mcp"
    }
  }
}

Mode 3: HTTP Streaming - Public Service

Use the hosted endpoint where each user provides their own API key:

{
  "mcpServers": {
    "keywords-ai": {
      "url": "https://keywordsai-mcp.vercel.app/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_KEYWORDS_AI_API_KEY"
      }
    }
  }
}

Available Tools

Category Tools
Logs list_logs, get_log_detail
Traces list_traces, get_trace_tree
Customers list_customers, get_customer_detail
Prompts list_prompts, get_prompt_detail, list_prompt_versions, get_prompt_version_detail

Filter Examples

All list tools support powerful filtering:

{
  "cost": {"operator": "gt", "value": [0.01]},
  "model": {"operator": "", "value": ["gpt-4"]},
  "customer_identifier": {"operator": "contains", "value": ["user"]},
  "metadata__session_id": {"operator": "", "value": ["abc123"]}
}

Operators: "" (equal), not, lt, lte, gt, gte, contains, icontains, startswith, endswith, in, isnull
