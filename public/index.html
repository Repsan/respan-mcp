Keywords AI MCP Server

This is a Model Context Protocol (MCP) server for Keywords AI.

API Endpoint

The MCP server is available at: /api/mcp

Usage with Cursor

Add this to your ~/.cursor/mcp.json:

{
  "mcpServers": {
    "keywords-ai": {
      "url": "https://your-deployment.vercel.app/api/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_KEYWORDS_AI_API_KEY"
      }
    }
  }
}

Available Tools

  • Logs: list_logs, get_log_detail
  • Traces: list_traces, get_trace_tree
  • Users: list_customers, get_customer_budget_detail
  • Prompts: list_prompts, get_prompt_detail, etc.
